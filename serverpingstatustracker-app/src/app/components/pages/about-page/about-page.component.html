About "Server Ping Status Tracker" App

Server Ping Status Tracker is a web-based application built with Angular and Spring Boot REST API (connected to a MySQL database) that allows users to monitor servers availability in a local network by performing ping tests and displaying their online/offline status. Users can also perform CRUD (Create, Read, Update, Delete) operations on Servers.
